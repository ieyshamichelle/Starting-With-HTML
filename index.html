The Basics of the Web and HTML

World Wide Web: Collection of HTML. HTML is HyperText Markup Language. Any files can be found or exist on the web. (Plain Text, HTML, Images, Videos, Music)

The Major pieces of the web consist of:
Browser ---> Internet ---> Servers

The request made from the internet to servers are called "HTTP". The server respond with files that the browser displays. Servers are computers that host the files that make up the World Wide Web.

HTML: HyperText Markup Language
HTML is made up of text content, mark up, refrences, and links to other web pages.
